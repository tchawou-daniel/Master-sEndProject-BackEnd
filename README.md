# Master-sEndProject
## Installation

```bash
$ npm i -g @nestjs/cli
$ yarn install
Create a .env.stage.dev file in the root of the project and configure it
```

## Running the app

```bash
# development
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```

## Lint
```bash
# linter
$ yarn lint
```
